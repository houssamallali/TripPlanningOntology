Trip Planning Ontology Documentation

This project contains an ontology for modeling online trip planning services, along with instance data and documentation.

Overview

The Trip Planning Ontology models:

  • Transportation modes and lines
  • Trip segments and itineraries
  • Weather conditions
  • Pricing models
  • Accommodations
  • Trip planning services and operations

Key Components

The ontology contains:

  • Classes: Trip Plan, Trip Segment, Transportation Mode, Weather, etc.
  • Object Properties: hasWeather, hasConnection, usesTransportationLine, etc.
  • Data Properties: price, duration, temperature, etc.
  • Named Individuals: Sample trip segments, transportation lines, pricing models, etc.

Project Files

You can find the following files in this project:

  • ../ontology/ontology.rdf - Ontology in RDF/XML format
  • ../instances/trip_instances.ttl - Sample instances demonstrating the ontology
